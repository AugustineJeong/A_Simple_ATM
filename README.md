# A_Simple_ATM

**Steps to run the program:**
1. Clone repository.
2. **cd** into **/A_Simple_ATM** directory.
3. Enter command:
```
./gradlew --console plain run
```
<br/>

**Steps to run the unit tests:**
<br/>
*(Tests only implemented for methods that don't require user input.)*
1. Clone repository.
2. **cd** into **/A_Simple_ATM** directory.
3. Enter command:
```
./gradlew test
```
